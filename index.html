CSS Topic: Padding vs. Margin

It's important to know the difference when formatting your page. You want the right things to be in the right places. To demonstrate, we will start out with some content wrapped in a blue box. Through this reading you will be able to see the effects of margin and padding to this content.

Margin

The CSS margin properties are used to create space around elements, outside of any defined borders. With CSS, you have full control over the margins. There are four different margin properties to use: margin-left, margin-right, margin-top, and margin-bottom. These can be seen in the diagram below.

Margin Short-hand

To shorten the code, it is possible to specify all the margin properties in one property. In this property you can specify up to four values to help the code know where to put things. Use the following list to see what each value means:

  • 4-Values
    • margin: 25px 50px 75px 100px;
      • top margin is 25px
      • right margin is 50px
      • bottom margin is 75px
      • left margin is 100px
  • 3-Values
    • margin: 25px 50px 75px;
      • top margin is 25px
      • right and left margin are 50px
      • bottom margin is 75px
  • 2-Values
    • margin: 25px 50px;
      • top and bottom margin are 25px
      • right and left margin are 50px
  • 1-Value
    • margin: 25px;
      • all four margins are 25px

Margin Collapse

Top and bottom margins of elements are sometimes collapsed into a single margin that is equal to the largest of the two margins. This does not happen on left and right margins! Only top and bottom margins!
